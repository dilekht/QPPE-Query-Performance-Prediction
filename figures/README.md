# Figures Directory

This directory contains visualizations generated by the QPPE visualization script.

## Generated Figures

After running the visualization script, you will find the following files:

| File | Description |
|------|-------------|
| `1_class_distribution.png` | Training data class balance (bar + pie chart) |
| `2_execution_time_distribution.png` | Execution time histograms by class |
| `3_feature_importance.png` | Feature importance scores from ML model |
| `4_confusion_matrix.png` | Prediction accuracy matrix |
| `5_model_performance_metrics.png` | Precision, Recall, F1 per class |
| `6_query_template_analysis.png` | Query diversity analysis |
| `7_feature_correlations.png` | Feature correlation heatmap |
| `8_learning_curve.png` | Model learning over sample size |
| `9_performance_improvement.png` | Baseline vs QPPE comparison |
| `10_training_summary_dashboard.png` | Complete summary dashboard |

## Generating Figures

```bash
python scripts/4_visualize_results.py \
    --db tpch \
    --user postgres \
    --password YOUR_PASSWORD \
    --output-dir ./figures
```

## Requirements

- Trained model file (`qppe_model_enhanced.pkl`)
- Training data in database
- matplotlib, seaborn, pandas installed
